What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces label pieces of content such as "this is a heading","this is paragrapgh","this is a link",etc

What is an HTML element?

An HTML element id defined by a start tag,some content ,and an end tag:

<tagname> Content goes here...</tagname>

The HTML element is everything from the start tag to the end tag:

<h1>My First Heading</h1>

<h1>My first paragraph.</h1>

Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br> none none

Web Browsers

The purpose of Web browser (Chrome, Edge,Firefox,Safari) is to read HTML documents and display them correctly.

The browser does not display HTML tag , but uses them to determine how to display the documents.
